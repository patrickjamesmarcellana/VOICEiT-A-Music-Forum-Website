ABOUT VOICEiT

Welcome to VOICEiT!

VoiceIt is a music forum website that allows users to share their thoughts and opinions on various topics related to music. Users can create posts, comment on other posts, cast upvotes and downvotes, and so much more! Explore the different music topics to see whats new and hot within specific genres. Contribute and engage with fellow users to talk about your favorite artists, albums, and songs. Start a conversation by posting a voice-it today!

Dependencies

VoiceIt was written in and requires Node.js, and it uses MongoDB for its database integration. It also uses jQuery for DOM manipulation and rich-text editing.

The following is a list of libraries and packages used to make VoiceIt possible.
Dependency Remark
mongoose Object data modeling library for MongoDB.
bcrypt For hashing user passwords.
passport For user authentication.
passport-local For local user authentication.
express Node.js web application framework the application was written with.
express-session For managing user sessions.
connect-mongo For managing user sessions.
body-parser To parse request bodies.
method-override To use other HTTP request methods like PATCH.
dotenv To load custom environment variables in local instances.
js-cookie For handling cookies.
multer For handling file uploads.
