About VOICEiT


Welcome to VOICEiT!

VOICEiT is a website that allows users to share their thoughts and opinions on various topics related to music. Users can create posts, comment on other posts, upvote or downvote them, and so much more!

Explore the subforums in order to see whats new or hot within specific genres, contribute, or talk about your favorite artists, albums, and songs!

So, what are you waiting for? Sign up today and join the VOICEiT community!

Dependencies

VoiceIt was written in and requires Node.js, and uses MongoDB for the database. It also uses jQuery for DOM manipulation and rich text.

The following is a list of libraries and packages used to make VoiceIt possible.

Dependency Remark
mongoose Object data modeling library for MongoDB.
bcrypt For hashing user passwords.
passport For user authentication.
passport-local For local user authentication.
express Node.js web application framework the application was written with.
express-session For managing user sessions.
connect-mongo For managing user sessions.
body-parser To parse request bodies.
method-override To use other HTTP request methods like PUT.
dotenv To load custom environment variables in local instances.
js-cookie Handles cookies.
multer For handling file uploads.
